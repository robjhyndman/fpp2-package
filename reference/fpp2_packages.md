# List all packages loaded by fpp2

List all packages loaded by fpp2

## Usage

``` r
fpp2_packages(include_self = FALSE)
```

## Arguments

- include_self:

  Include fpp2 in the list?

## Value

A character vector of package names.

## Examples

``` r
fpp2_packages()
#> [1] "cli"        "crayon"     "expsmooth"  "fma"        "forecast"  
#> [6] "ggplot2"    "magrittr"   "purrr"      "rstudioapi"
```
